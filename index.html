CSS GRID

What`s css grid?

  • CSS Grid Layout is a two-dimensional grid-based layout system
  • It was created by CSS Working Group in 2017

Browser support

browser support reached 94% in 2020

Properties for the grid container

display

              
                .container {
                  display: grid | inline-grid;
                }

grid-template-columns

grid-template-rows

  
    .container {
      grid-template-columns: [first] 40px [line2] 50px auto;
      grid-template-rows: 25% 100px auto;
    }

grid-template-areas

column-gap

row-gap

  
    .container {
      column-gap: 10px;
      row-gap: 15px;
    }

justify-content

  
    .container {
      justify-content: start | end | center | space-between
    }

align-content

  
    .container {
      align-content: start | end | center | space-between
    }

Properties for the grid items

Note

float, display: inline-block, display: table-cell, vertical-align properties have no effect on a grid item
